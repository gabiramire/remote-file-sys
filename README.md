# remote-file-sys

## Para rodar:

- Dentro de java-lib, rodar:

 ``` mvn compile ```
    
- Para rodar o server:

 ``` mvn exec:java -Dexec.mainClass=remote.ServerMain ```

- Para rodar o Client:

 ``` mvn exec:java -Dexec.mainClass=remote.ClientMain ```
